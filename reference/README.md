# Reference Sources

`reference` stores copied source snapshots and protocol references from external projects.

Rules:
- Do not put Redou runtime code here.
- Do not keep nested Git repositories here.
- Do not compile or import these files from Redou mainline code.
- Each external source snapshot must include a matching `README.redou.md` explaining origin, copy time, and usage boundaries.

Future Claude Code, Pi, or Hermes reference snapshots should be placed under this directory with the same metadata and no nested `.git` directory.
